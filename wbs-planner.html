接続設定

API トークンは Atlassian アカウント設定（Security → API tokens）で発行できます。このブラウザのみに保存されます。

CORS エラーが発生する場合に設定します。python3 scripts/jira-proxy.py を別ターミナルで起動してください。

担当者マッピング

プロジェクトキーを設定して「JIRA ユーザー取得」を押すと、担当者と JIRA アカウントを紐付けられます。
